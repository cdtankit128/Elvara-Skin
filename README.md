# Elvara Skin Shopify Theme

Elvara Skin is a high-end, luxury-focused Shopify storefront theme. It has been meticulously custom-developed to provide a minimalist, premium e-commerce experience tailored for skincare and beauty brands.

## Project Description

Developed to serve as an industry-leading storefront, the Elvara Skin theme focuses on pixel-perfect layouts, modern dynamic aesthetics, and top-tier user experiences. The overarching design language embodies a cohesive, minimalist high-end brand aesthetic.

### Key Contributions & Technical Highlights

- **Custom Theme Development**: Re-architected standard Shopify structures into a sleek, modular design system with specialized sections like `video-showcase`, `testimonials-premium`, and `product-benefits`.
- **Performance Tuning**: Significantly enhanced Core Web Vitals, specifically aiming at Cumulative Layout Shift (CLS) reduction and optimal asset loading to ensure immediate, jitter-free page rendering.
- **Conversion-Rate Optimization (CRO)**: Implemented streamlined checkout flows, sticky CTAs, and trust-building UI elements designed to seamlessly guide the customer from discovery to purchase.
- **Dynamic Aethestics**: Employs smooth micro-animations, glassmorphism, and meticulously tuned CSS to capture a premium feel.

## Tech Stack

- **Core**: Shopify Liquid, HTML5, Vanilla JavaScript.
- **Styling**: Vanilla CSS engineered for maximal flexibility and fast execution.
- **Config**: JSON templates.

## Local Development

If you'd like to run or test this theme locally:

1. Ensure you have the [Shopify CLI](https://shopify.dev/docs/apps/tools/cli) installed.
2. Clone the repository:
   ```bash
   git clone https://github.com/cdtankit128/Elvara-Skin.git
   ```
3. Navigate to the project directory and start the local Shopify server:
   ```bash
   cd Elvara-Skin
   shopify theme dev
   ```
